2.1 猜数游戏：一次猜测

目标

  • 生成一个1到100间的随机数
  • 提示玩家输入一个猜测
  • 猜完之后，程序会提示猜测是太小了还是太大了
  • 如果猜测正确,那么打印出一个庆祝信息，程序退出

上手编写

  1. 新建项目：cargo new guessing_game
  2. 打开src/main.rs

use std::io;

fn main() {
    println!("欢迎进入猜数游戏");
    
    println!("请输入一个数字：");
    
    let mut guess = String::new();

    io::stdin().read_line(&mut guess).expect("读取失败");

    println!("你输入的数字是：{}", guess);

}

输入

要获取用户输入内容并打印输出，我们需要使用io库，而io库又在标准库std(Standard)中。在一般情况，Rust会自动导入一定模块到程序的作用域中，这个操作称为prelude，但是如果我们要使用的库不包含在prelude中则需要使用use语句显式导入相应的库，即：

1

use std::io;

转到下方的io语句，可见有多个表达式通过.成员访问符相连而成，让我们分开解析它：

1

io::stdin()

这个表达式的意思是调用io库中的的stdin()关联函数，它会返回一个Stdin类型的实例，这个实例会被用作 句柄（handle） 来处理终端中的标准输入。

Note

句柄是操作系统用来标识各类资源的无特殊含义整数标识，程序打开文件、窗口、网络连接、内存对象后系统会分配唯一句柄，程序通过传递句柄就能操作对应资源，不用直接访问底层资源地址，资源释放时句柄随之失效。

指针直接存储内存真实地址，可随意加减偏移、访问任意内存；句柄是系统提供的抽象编号，不暴露真实内存地址，只能传给系统API操作资源，不能直接解引用，安全性更高。指针失效会造成野指针、内存越界崩溃，句柄失效仅会操作失败，不会乱访问内存；指针多用于用户态直接操作内存，句柄用于管理文件、窗口、内核对象等系统资源。

如果上面我们没有使用use导入语句，那么这里我们也可以写成：

1

std::io::stdin()

接下来我们调用了io::stdin()这个实例当中的read_line方法，注意因为此处调用的是实例中的方法，所以使用.点运算符，详细说明见下方赋值部分的 NOTE 标记。

通过read_line方法，我们可以把用户输入的内容放到一个字符串当中，因此这里的参数需要一个 可变字符串 ，而这里的参数&mut guess前带有取地址符号&，这告诉我们这里参数是以引用的形式传入的（也就是所谓的 方法的参数是按引用进行传递的 ）

Note

将鼠标悬浮在read_line方法上我们可以看到它的说明，其中标注了该方法有一个返回值io::Result<usize>

io::Result 是 std::io 模块下封装好的结果类型，<usize> 代表成功时返回的值为无符号整数（一般代表读取 / 写入的字节数量）。该类型用于 IO 相关操作，分为两种状态： Ok (结果) 表示读写成功并顺带返回结果， Err (失败原因) 表示读写失败并携带 IO 异常信息，通过该返回值，该表达式是否操作成功可以被轻易的得知。

接下来的ecpect()这个异常捕获方法就是通过读取read_line的返回值来工作，当read_line的返回值为Err，ecpect()会中断当前的程序并显示传入的字符串信息；当read_line的返回值为Ok，ecpect()会提取返回值当中的结果，并将其返回给用户。

如果我们去掉ecpect()直接编译程序，这会看到编译器抛出的警告，从这一点可以看出Rust的安全性。

来到最后的输出语句：

1

println!("你输入的数字是：{}", guess);

此处的{}是一个占位符，对应着后面的变量值；当一个println!语句中有多个占位符和变量时，占位符会按照从前往后的顺序一一对应后面的变量。

赋值

在 Rust 当中，我们使用let语句来为变量赋值，例如：

1
2

let one = 1;
let num = one;

上面的语句会把one和num两个变量都赋值为1。

在Rust中的一大特点就是它的变量默认是 不可变的（immutable） ，想要让变量可变，则需要在声明变量时加上mut关键字，例如：

1
2
3

let mut num = 1;
let one = 1;	//immutable
num = 2;

让我们回到实例当中的赋值语句：

1

let mut guess = String::new();

这里使用等号将可变变量guess绑定了右面的表达式所返回的的值。

String::new()会返回一个字符串的新的实例，在 Rust 中，字符串类型即为String，而new是String类型的一个关联函数，这里使用:: （路径分隔符） 来取String类型的new这个项。

在Rust当中，很多类型都有一个new()关联函数，您可以用它来创建一个对应的新的实例。

Note

::是Rust路径分隔符，用于模块、类型等静态命名空间，无需实例即可访问关联函数、常量、枚举变体（不针对任何一个定义出来的实例）；区别于.，后者仅操作实例对象。
