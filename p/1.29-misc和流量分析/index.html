wireshark的使用

  1. 过滤器的使用：Wireshark过滤器写法总结 - willingtolove - 博客园
  2. Ctrl+F的使用：正常情况下选字符串、除非搜文件选十六进制值查文件头。
  3. 右键单个数据包使用追踪HTTP流可以查看更详细的数据流（如相关回包）

分析流程

  1. 统计-协议分级：确定重点分析的流量：

    • 常规流量：http

    • 重点关注流量（出现且占比较大）：

    • https 找https解密密钥文件

    • dns 成段数据 base64加密

    • telnet 明文

    • icmp data

    • ftp 记录交互过程

    • ftp-data 记录传递的文件数据

    • 802.1x WiFi协议 爆破WiFi密码

  2. 看看第一条的source是谁，谁就是黑客：

    http && ip.addr == 黑客IP

  3. 找比较特殊的流量数据：

    • 首先变化的流量
    • 最后的流量
    • 包含命令的流量
    • 包含文件/数据的流量

目标（基础）：

  • 黑客ip是什么？
  • 从sql注入中获得了什么？
  • 最后进行了使用了什么攻击方式？
